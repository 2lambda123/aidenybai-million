## Thinking in Million

Million is based on the following principles:

- __Performance__: Million.js is designed to be as fast as possible. This is achieved through a number of optimizations, including the use of blocks and the avoidance of unnecessary DOM updates.

- __Simplicity__: Million.js is designed to be easy to use. It is a drop-in replacement for React, so you can start using it right away without having to learn a new API.

- __Flexibility__: Million.js is designed to be flexible. It can be used to build a wide variety of UIs, from simple to complex.


The philosophy behind Million.js is to provide a high-performance, easy-to-use, and flexible virtual DOM for React applications.





## Conclusion

Million.js is a powerful virtual DOM (cannot stress this enough) that can be used to build fast and performant web applications. 

By thinking in blocks, you can write more efficient and performant frontend code with Million.js.

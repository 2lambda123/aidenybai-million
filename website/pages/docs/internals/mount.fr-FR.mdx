# `mount(){:jsx}`

**Syntaxe:** `mount(Block, el){:jsx}`\
**Exemple:** `mount(block, document.getElementById('root')){:jsx}`

La fonction `mount` est utilisée pour monter un bloc sur un élément DOM (imaginez `ReactDOM.render()` dans React). Elle prend deux arguments : le bloc à monter et l'élément DOM sur lequel le monter.

```jsx
import { block, mount } from 'million';

const display = block(({ text }) => {
  return <p>{text}</p>;
});

const element = mount(display, document.getElementById('root'));
```

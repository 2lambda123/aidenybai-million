# `mapArray(){:jsx}`

**Syntaxe:** `mapArray(Block[]){:jsx}`\
**Exemple:** `mapArray([block, block, block]){:jsx}`

La fonction `mapArray` est utilisée pour créer une liste de blocs. C'est la meilleure façon de rendre une vue dérivée de données semblables à un tableau. Lorsque le tableau change, `mapArray(){:jsx}` met à jour ou déplace les éléments dans le DOM plutôt que de les recréer. Regardons un exemple :

```jsx
import { block, patch, mapArray } from 'million';

const oldList = [1, 2, 3];
const newList = [3, 2, 1];
const list = block(({ item }) => {
  return <div>{item}</div>;
});

// updates list efficiently (only 2 moves instead of 3 updates)
patch(
  document.body,
  mapArray(oldList.map((item) => list({ item }))),
  mapArray(newList.map((item) => list({ item }))),
);
```

# `stringToDOM(){:jsx}`

**Syntaxe:** `stringToDOM(content, svg){:jsx}`\
**Exemple:** `stringToDOM('<div>Hello World</div>'){:jsx}`

La fonction `stringToDOM` prend une chaîne HTML ou SVG et renvoie un objet DOM. Cela est utile pour créer des éléments DOM à partir de chaînes.

```jsx
import { stringToDOM } from 'million';

const dom = stringToDOM('<div>Hello World</div>');

dom.innerHTML; // 'Hello World'
dom.tagName; // 'DIV'

const svg = stringToDOM('<g><rect /></g>'); // creates SVG!
```

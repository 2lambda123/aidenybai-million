# `字符串转DOM(){:jsx}`

**句法:** `stringToDOM(content, svg){:jsx}`\
**例子:** `stringToDOM('<div>Hello World</div>'){:jsx}`

`stringToDOM` 函数接受 HTML 或 SVG 字符串并返回 DOM 对象。 这对于从字符串创建 DOM 元素非常有用。

```jsx
import { stringToDOM } from 'million';

const dom = stringToDOM('<div>Hello World</div>');

dom.innerHTML; // 'Hello World'
dom.tagName; // 'DIV'

const svg = stringToDOM('<g><rect /></g>'); // creates SVG!
```

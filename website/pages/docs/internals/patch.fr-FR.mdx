# `patch(){:jsx}`

**Syntaxe:** `patch(oldBlock, newBlock){:jsx}`\
**Exemple:** `patch(block1, block2){:jsx}`

La fonction `patch` est utilisée pour rerendre un bloc avec un autre bloc. Le `oldBlock`  est le bloc qui sera rerendu, et le `newBlock` représente la nouvelle version du DOM.

<Callout>
  Les blocs doivent être dérivés de la même fonction lors de l'utilisation de `patch()`. Cela garantit que les performances ne sont pas impactées négativement.
</Callout>

```jsx
import { block, mount, patch, fragment } from 'million';

const display = block(({ text }) => {
  return <p>{text}</p>;
});

// we will patch against this block for updates
const main = display({ text: 'Hello' });

mount(main, document.getElementById('root'));

patch(main, display({ text: 'World' }));

const bigDisplay = block(({ text }) => {
  return <h1 style={{ color: 'red' }}>{text}</h1>;
});

patch(main, bigDisplay({ text: 'World' })); // inefficient, but works
```

# `山(){:jsx}`

**句法:** `mount(Block, el){:jsx}`\
**例子:** `mount(block, document.getElementById('root')){:jsx}`

`mount` 函数用于将块安装到 DOM 元素 (想象一下 React 中的 `ReactDOM.render()`). 它需要两个参数：要挂载的块和要挂载的 DOM 元素。

```jsx
import { block, mount } from 'million';

const display = block(({ text }) => {
  return <p>{text}</p>;
});

const element = mount(display, document.getElementById('root'));
```

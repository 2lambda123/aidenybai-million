# `block()`

**Syntax:** `block((props) => vnode)`\
**Example:** `block((props) => <div>{props.foo}</div>)`

The `block` function instantiates a `Block` (_a stateless "component"_). It accepts a function with a props object parameter and returns a `VNode`.

## Rules of `block()`

Anatomy of a `Hole`

##### `props` is an immutable object with primitive or `Block` values.

```jsx
someBlock({
  one: '1', // ✅
  two: 1 + 1 // ✅
  three: true // ✅
  four: Date.now() // ✅
  five: { imposter: true } // ❌
  six: new Date() // ❌
});
```

##### Values of `props` may not be interpolated with other values.

```jsx
const block = createBlock((props) => {
  const { favorite } = props.favorite; // ❌
  <div className={props.className /* ✅ */}>
    {props.hello /* ✅ */}
    {Date.now() /* ✅ */}
    <button
      onClick={() => {
        console.log(props.world); /* ❌ */
      }}
    >
      {props.count + 1 /* ❌ */}
      {props.foo.toString() /* ❌ */}
    </button>
  </div>;
});
```

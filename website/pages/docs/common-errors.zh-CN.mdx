# 常见错误

您来到这里可能是因为您在将现有项目与 Million.js 集成时遇到问题。

## MobX 集成

- 升级到 MobX v6
- 删除装饰器

## 双宏

Million.js 无法与 `twin.macro` 一起使用，因为它们的 JSX 转换覆盖。
